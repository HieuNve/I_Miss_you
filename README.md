## Cài Đặt

Hướng dẫn cách cài đặt và chạy dự án:

1. Clone dự án:
    ```bash
    git clone https://github.com/HieuNve/I_Miss_you.git
    ```

2. Di chuyển vào thư mục dự án:
    ```bash
    cd I_Miss_you
    ```

3. Cài đặt các thư viện yêu cầu:
    ```bash
    pip install -r requirements.txt
    ```

4. Chạy dự án:
    ```bash
    python main.py
    ```
